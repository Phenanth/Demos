# 智能家居管理系统

## 源代码核心文件说明

`index.html`为总入口，`app.js`为核心逻辑，`static/css/index.css`为自写样式，`README.md`为说明。

使用的模块存放于`static/css`各目录中。

## 功能说明

模拟了智能家居控制系统根据当前天气指标（温度、湿度、PM2.5值）对家中电器的开关。支持用户点击互动各类电器。

此外还通过HTML5实现了各模块出现与否与出现位置根据页面大小的动态变化。

各功能区说明见总页面内。
